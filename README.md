# Tutorial for XDevelopers on GPS use

This sample application uses the HTML5 geolocation API
The GPS coordinates are more exact if a GPS device is present and enabled in the device

This project uses the standard PhoneGap directory setup.

www - Project directory
www/css - Cascade Stylesheet
www/img - Images used in the project
www/js - JavaScript files
config.xml - PhoneGap configuration file
icon/png - Application icon
index.html - HTML main application file



Built with PhoneGap Build, utilizes HTML5 geolocation API
